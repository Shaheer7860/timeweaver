                    BUILD INSTRUCTIONS
===============================================================================

The build script (build.bat) has been updated to work with this structure:

1. Compiles SQLite from database/ folder
2. Compiles C++ from src/ folder
3. Links everything together
4. Creates TimeWeaver.exe in root folder

To build:
```
build.bat
```

To run:
```
TimeWeaver.exe
```
